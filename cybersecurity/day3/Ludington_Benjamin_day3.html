Day 3 - Configuring a Secure Linux Environment for WordPress Deployment Using SSH, MySQL, and Apache

Commands used to create the Linux user:

  1. sudo adduser wordpress
  2. sudo usermod -aG sudo wordpress
  3. groups wordpress : returned -->wordpress : wordpress sudo users
Screenshots showing the .ssh directory and authorized_keys file
Commands used to create the WordPress database and user
  1. mysql
  2. CREATE DATABASE wordpress DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
  3. CREATE USER 'wp_user'@'localhost' IDENTIFIED BY 'StrongPasswordHere';
  4. GRANT SELECT, INSERT, UPDATE, DELETE, CREATE, DROP, ALTER, INDEX
    ON wordpress.* TO 'wp_user'@'localhost';
  5. FLUSH PRIVILEGES;
  6. EXIT;
